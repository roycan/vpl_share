VPL Assignment Templates

Ready-to-use templates for creating effective programming assignments

These templates provide a starting point for creating effective VPL assignments. Each template includes the necessary files and scripts for automated evaluation. Simply copy the code and customize it to fit your specific assignment requirements.

Filter by language:

Basic Python Assignment

Python

A simple template for Python assignments that test basic input/output functionality.

Required Files

vpl_run.sh
vpl_evaluate.sh
vpl_test.cases

vpl_run.sh

#!/bin/bash python3 $VPL_SRCFILE

vpl_evaluate.sh

#!/bin/bash # Load common script and grade functions . vpl_environment.sh # Run the program with test cases cat > vpl_execution << "END_SCRIPT" #!/bin/bash python3 $VPL_SRCFILE < vpl_test.cases END_SCRIPT # Make the execution script executable chmod +x vpl_execution # Run the test cases and capture output ./vpl_execution > student_output.txt # Load the expected output cat > expected_output.txt << "END_OUTPUT" Sum: 30 Sum: 70 Sum: 0 END_OUTPUT # Compare student output with expected output DIFF=$(diff -w student_output.txt expected_output.txt) # Grading logic VPL_GRADEMAX=100 if [ -z "$DIFF" ]; then # Perfect match VPL_GRADE=100 COMMENT="Perfect! Your program produces the correct output." else # Some differences VPL_GRADE=0 COMMENT="Your program output doesn't match the expected output." echo "<|--" echo "$DIFF" echo "--|>" fi # Display feedback echo "Grade: $VPL_GRADE/$VPL_GRADEMAX" echo -e "$COMMENT"

Customization Tips:

  • Adjust the grading criteria based on the difficulty of the assignment

Basic Java Assignment

Java

A template for simple Java assignments with compilation and execution.

Required Files

vpl_run.sh
vpl_evaluate.sh
vpl_test.cases

vpl_run.sh

#!/bin/bash # Compile Java file javac *.java # Run Java program (assumes Main class with main method) java Main

vpl_evaluate.sh

#!/bin/bash # Load common script and grade functions . vpl_environment.sh # Compile Java file first javac *.java if [ $? -ne 0 ] ; then echo "Compilation error" exit 0 fi # Run the program with test cases cat > vpl_execution << "END_SCRIPT" #!/bin/bash java Main < vpl_test.cases END_SCRIPT # Make the execution script executable chmod +x vpl_execution # Run the test cases and capture output ./vpl_execution > student_output.txt # Load the expected output cat > expected_output.txt << "END_OUTPUT" Sum: 30 Sum: 70 Sum: 0 END_OUTPUT # Compare student output with expected output DIFF=$(diff -w student_output.txt expected_output.txt) # Grading logic VPL_GRADEMAX=100 if [ -z "$DIFF" ]; then # Perfect match VPL_GRADE=100 COMMENT="Perfect! Your program produces the correct output." else # Some differences VPL_GRADE=0 COMMENT="Your program output doesn't match the expected output." echo "<|--" echo "$DIFF" echo "--|>" fi # Display feedback echo "Grade: $VPL_GRADE/$VPL_GRADEMAX" echo -e "$COMMENT"

Example Student Solution (for reference)

import java.util.Scanner; public class Main { public static void main(String[] args) { Scanner scanner = new Scanner(System.in); while (scanner.hasNextLine()) { String line = scanner.nextLine(); String[] nums = line.split(" "); int a = Integer.parseInt(nums[0]); int b = Integer.parseInt(nums[1]); System.out.println("Sum: " + (a + b)); } scanner.close(); } }

Assignment Description Example:

Create a Java program that reads pairs of integers from input and outputs their sum. The program should process each line of input separately and output the sum for each pair.

Customization Tips:

  • Modify the expected output to match your specific assignment requirements
  • If the student solution uses a different class name, update the vpl_run.sh script
  • Add additional test cases to verify edge cases or error handling

Basic C/C++ Assignment

C/C++

A template for C/C++ assignments with compilation and execution.

Required Files

vpl_run.sh
vpl_evaluate.sh
vpl_test.cases

vpl_run.sh

#!/bin/bash # Compile C++ file g++ -o program *.cpp # Run program ./program

vpl_evaluate.sh

#!/bin/bash # Load common script and grade functions . vpl_environment.sh # Compile C++ file first g++ -o program *.cpp if [ $? -ne 0 ] ; then echo "Compilation error" exit 0 fi # Run the program with test cases cat > vpl_execution << "END_SCRIPT" #!/bin/bash ./program < vpl_test.cases END_SCRIPT # Make the execution script executable chmod +x vpl_execution # Run the test cases and capture output ./vpl_execution > student_output.txt # Load the expected output cat > expected_output.txt << "END_OUTPUT" Sum: 30 Sum: 70 Sum: 0 END_OUTPUT # Compare student output with expected output DIFF=$(diff -w student_output.txt expected_output.txt) # Grading logic VPL_GRADEMAX=100 if [ -z "$DIFF" ]; then # Perfect match VPL_GRADE=100 COMMENT="Perfect! Your program produces the correct output." else # Some differences VPL_GRADE=0 COMMENT="Your program output doesn't match the expected output." echo "<|--" echo "$DIFF" echo "--|>" fi # Display feedback echo "Grade: $VPL_GRADE/$VPL_GRADEMAX" echo -e "$COMMENT"

Customization Tips:

  • For C programs, change the compiler command from g++ to gcc

  • Check for memory leaks using Valgrind by adding it to the evaluation script
